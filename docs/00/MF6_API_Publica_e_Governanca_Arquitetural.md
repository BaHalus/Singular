# MF6 — API Pública e Governança Arquitetural

Status: Normativo
Versão: 2.0 (rascunho)

Este documento estabelece a API Pública da SINGULAR e seu modelo de governança arquitetural.

## Governança Arquitetural

A governança é composta por:

- Invariantes Arquiteturais (IA)
- Requisitos Arquiteturais (RA)
- Testes de Conformidade Arquitetural (AT)
- Processo de Evolução Arquitetural (PE)

Os capítulos IA, RA, AT e PE constituem a referência normativa da arquitetura da SINGULAR.
