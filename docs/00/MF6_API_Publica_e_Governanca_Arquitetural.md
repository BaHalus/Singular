# MF6 — API Pública e Governança Arquitetural

Status: Normativo
Versão: 2.0 (rascunho)

## Objetivo

Este módulo define a API Pública da SINGULAR, os contratos arquiteturais obrigatórios e as regras de governança que disciplinam a evolução do sistema.

## Escopo

O MF6 estabelece:

- a organização da API Pública;
- os contratos públicos da plataforma;
- os Invariantes Arquiteturais (IA);
- os Requisitos Arquiteturais (RA);
- os Testes de Conformidade (AT);
- o Processo de Evolução Arquitetural (PE).

## Governança Arquitetural

Toda implementação da SINGULAR deverá permanecer compatível com este documento. Em caso de conflito entre implementação e especificação, prevalece a especificação.

A governança arquitetural é composta por:

- Invariantes Arquiteturais (IA)
- Requisitos Arquiteturais (RA)
- Testes de Conformidade Arquitetural (AT)
- Processo de Evolução Arquitetural (PE)

Os capítulos IA, RA, AT e PE constituem a referência normativa da arquitetura da SINGULAR e serão expandidos progressivamente nas próximas revisões deste módulo.
