# SINGULAR 0.9.0-alpha.1

Data da baseline: 2026-07-11.

## Visão geral

Esta versão congela a primeira Alpha mobile utilizável da SINGULAR. O objetivo é oferecer uma referência estável para testes reais antes da campanha de distribuição PWA.

## O que já pode ser testado

- abrir `mobile.html`;
- criar e editar um personagem;
- alternar entre os modos Criação e Mesa;
- registrar alterações transitórias de PV e PF;
- usar as seções Alpha de traços, idiomas, culturas, perícias, técnicas, ataques, magias, poderes, equipamentos e carga;
- salvar, fechar, restaurar, exportar e importar uma ficha;
- validar persistência e comportamento mobile com a suíte automatizada.

## Garantias arquiteturais

- o motor calcula;
- o schema declara;
- a aplicação orquestra;
- a UI apresenta e coleta intenção;
- a persistência guarda snapshots;
- existe um único composition root mobile;
- esta release não introduz PWA, backend, sincronização ou pipeline paralelo.

## Validação

```bash
npm ci
npm test
npx playwright install chromium
npm run test:mobile-smoke
```

## Próxima campanha

A instalação PWA, o funcionamento offline e a publicação pública serão tratados separadamente após esta baseline.
