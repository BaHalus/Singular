# Guia de abertura e teste — 0.9.0-alpha.1

## Pré-requisitos

- Node.js 22.x;
- npm;
- navegador moderno.

## Preparação

```bash
git clone https://github.com/BaHalus/Singular.git
cd Singular
git checkout release/alpha-baseline-and-docs
npm ci
```

## Testes automatizados

```bash
npm test
npx playwright install chromium
npm run test:mobile-smoke
```

## Abertura local

Sirva a raiz do repositório por HTTP. Exemplo:

```bash
npx serve .
```

Abra a URL informada pelo servidor e acesse `/mobile.html`. Não abra o arquivo diretamente por `file://`, porque módulos ES exigem contexto HTTP em navegadores comuns.

## Checklist manual mobile

1. Abra `mobile.html` em viewport de celular.
2. Crie ou edite a identidade do personagem.
3. Edite atributos e ao menos uma seção estrutural no modo Criação.
4. Salve manualmente e confirme o autosave.
5. Feche e reabra a página; restaure a sessão.
6. Alterne para o modo Mesa.
7. Confirme que um controle estrutural fica bloqueado.
8. Altere PV e PF atuais.
9. Exporte a ficha e importe o arquivo exportado.
10. Confirme ausência de tela branca, perda silenciosa e erro não tratado no console.

## Proteção dos dados

Antes de trocar de aparelho, navegador ou perfil, exporte a ficha. Esta Alpha ainda não possui sincronização ou recuperação em nuvem.
