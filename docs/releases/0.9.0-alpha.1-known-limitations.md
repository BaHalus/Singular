# Limitações conhecidas — 0.9.0-alpha.1

Esta é uma versão Alpha destinada a validação funcional e arquitetural.

## Distribuição

- ainda não há instalação PWA;
- não há service worker nem garantia de funcionamento offline;
- não há URL pública oficial;
- não há APK ou aplicativo nativo.

## Dados

- os dados permanecem locais ao navegador e ao perfil utilizado;
- limpar os dados do navegador pode remover salvamentos locais;
- não há sincronização em nuvem, login ou recuperação remota;
- antes de trocar de navegador, aparelho ou perfil, o usuário deve exportar a ficha.

## Compatibilidade

- a validação automatizada principal usa Chromium;
- Safari/iPhone e outros navegadores ainda exigem checklist manual;
- navegadores antigos ou sem módulos ES não são suportados.

## Escopo funcional

- biblioteca modular e integração GCS continuam planejadas;
- acessibilidade, temas e refinamento visual ainda não representam o acabamento final;
- campos e capacidades avançadas de GURPS podem estar incompletos;
- esta baseline não promete compatibilidade de dados com versões futuras sem migração.

## Uso recomendado

Use cópias exportadas para testes importantes e registre problemas com navegador, aparelho, passos de reprodução e resultado observado.
