# SINGULAR

Sistema de ficha para GURPS 4ª Edição desenvolvido em um único arquivo HTML.

## Objetivos

- Mobile First
- Interface extremamente compacta
- Motor desacoplado da interface
- Schema declarativo
- Persistência local
- Biblioteca modular
- Importação GCS
- Temas
- Acessibilidade
- Alta performance

## Arquitetura

A aplicação é dividida em cinco camadas independentes:

1. Schema
2. Motor
3. Persistência
4. Importação/Exportação
5. Interface

Princípio central:

> O motor calcula. O schema declara. A UI apenas apresenta informações e coleta entrada do usuário.
