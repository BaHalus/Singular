# Invariantes — TraitControl

- `roll` é inteiro não negativo;
- `status` é recalculado a partir do teste;
- `multiplier` é recalculado a partir da tabela soberana;
- `penalty` é recalculada a partir do autocontrole;
- ajuste desconhecido não recebe valor mecânico;
- teste desconhecido não recebe multiplicador;
- serialização nunca transforma `raw` em autoridade.
