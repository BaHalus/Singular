# Filosofia da SINGULAR

A SINGULAR existe para reduzir a carga cognitiva do jogador.

Ela deve permitir que o usuário pense no jogo, e não na ficha.

## Princípios

- Mobile First sempre.
- Informação antes de decoração.
- Automação deve ser previsível.
- O usuário sempre entende por que um valor foi calculado.
- Nenhuma informação importante será escondida.
- Alta densidade de informação sem perda de legibilidade.
- O motor é soberano.
- A interface nunca calcula.
- Componentes são independentes.
- O código deve ficar mais simples com o tempo, nunca mais complexo.
